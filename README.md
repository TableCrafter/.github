# Welcome to TableCrafter

### Data-Driven Tools for WordPress Creators

We build a suite of powerful, developer-friendly plugins designed to transform how you work with data in WordPress. From beautiful card grids to advanced tables and high-conversion lead magnets, our ecosystem is built for performance and flexibility.

---

## 🛠️ The Ecosystem

### 🃏 [CardCrafter](https://github.com/TableCrafter/cardcrafter-data-grids)
**JSON → Beautiful Card Grids**
Transform any JSON data source into responsive, customizable card layouts. Perfect for team directories, portfolios, and product showcases.
*   **Key Features:** Secure Proxy, SWR Caching, Masonry Layouts.

### 📊 [Gravity Tables](https://github.com/fahdi/gravity-tables)
**Advanced Data Tables for Gravity Forms**
The premium solution for displaying, editing, and managing Gravity Forms entries on the frontend.
*   **Key Features:** Inline Editing, Advanced Filtering, Excel/CSV Export.

### 🧲 [KitLeads](https://github.com/fahdi/kit-leads-for-wp)
**Grand Slam Lead Magnets**
A lightweight, high-performance integration for Kit.com (ConvertKit). Capture leads with style and sync them instantly.
*   **Key Features:** Zero-Touch Setup, Fallback Handling, Developer API.

### 📦 [TableCrafter Core](https://github.com/TableCrafter/tablecrafter)
The foundational library powering our suite. Provides unified data fetching, state management, and UI components across all our plugins.

## 🚀 Our Philosophy
*   **Performance First:** We obsess over milliseconds.
*   **Security by Design:** We use strict validation and secure proxies (like `wp_safe_remote_get`).
*   **Developer Friendly:** Hooks, filters, and clean APIs for everything.
